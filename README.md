# 🐦 Flappy Bird 复刻版

这是一个使用 HTML5 Canvas 和纯 JavaScript 制作的 Flappy Bird 游戏复刻版本。

## 🎮 游戏特性

- **完整复现**：忠实还原了经典 Flappy Bird 的游戏机制
- **现代界面**：采用现代 CSS 设计，包含动画效果和响应式布局
- **多种控制方式**：支持鼠标点击和键盘空格键控制
- **分数系统**：实时分数显示和本地最高分记录
- **流畅动画**：平滑的小鸟物理效果和视觉反馈

## 📁 文件结构

```
bird/
├── index.html     # 游戏主界面
├── style.css      # 样式文件
├── game.js        # 游戏逻辑
└── README.md      # 项目说明
```

## 🚀 运行方法

### 本地运行

1. 直接在浏览器中打开 `index.html` 文件
2. 或使用本地服务器：
   ```bash
   # 使用 Python 3
   python -m http.server 8000
   
   # 或使用 Node.js 的 http-server
   npx http-server
   ```
3. 在浏览器中访问 `http://localhost:8000`

### GitHub Pages 部署

1. 将所有文件上传到 GitHub 仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择 `main` 分支作为源
4. 访问提供的 GitHub Pages URL 即可游玩

## 🎯 游戏说明

### 游戏目标
- 控制小鸟避开绿色管道
- 穿过管道之间的空隙获得分数
- 尽可能获得更高分数

### 操作方法
- **鼠标**：点击游戏画布让小鸟跳跃
- **键盘**：按空格键让小鸟跳跃

### 游戏规则
- 小鸟会自动下落，需要点击保持飞行
- 碰到管道或地面游戏结束
- 穿过管道即可获得 1 分
- 最高分自动保存到本地存储

## 🛠️ 技术实现

### 核心技术
- **HTML5 Canvas**：游戏渲染
- **原生 JavaScript**：游戏逻辑和物理引擎
- **CSS3**：现代界面设计和动画效果
- **LocalStorage**：分数持久化存储

### 游戏机制
- **物理系统**：重力模拟和跳跃机制
- **碰撞检测**：精确的小鸟与管道碰撞检测
- **随机生成**：管道高度随机变化增加挑战性
- **游戏状态管理**：准备、游戏中、结束三种状态

## 🎨 自定义修改

### 修改游戏参数
在 `game.js` 中可以调整以下参数：

```javascript
// 小鸟物理参数
this.bird.gravity = 0.5;        // 重力强度
this.bird.jump = -8;            // 跳跃力度

// 管道参数
this.pipeGap = 150;             // 管道间距
this.pipeSpeed = 2;             // 管道移动速度
this.pipeSpawnRate = 90;        // 管道生成频率
```

### 修改样式
在 `style.css` 中可以自定义：
- 颜色主题
- 动画效果
- 响应式布局
- 界面尺寸
